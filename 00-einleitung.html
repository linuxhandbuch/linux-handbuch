Einleitung

Willkommen im Kreativen Linux Handbuch. Dieses Nachschlagewerk richtet sich an Einsteiger ohne viel Linux-Erfahrung und erklärt die wichtigsten Befehle praxisnah, direkt und auf den Punkt.

Willkommen 01

Linux ist ein freies Betriebssystem, das auf sehr vielen Servern, aber auch immer mehr Desktop-Rechnern läuft. Anders als bei Windows oder macOS steuert man Linux oft über die Kommandozeile (das Terminal) statt über Klicks mit der Maus. Das wirkt am Anfang ungewohnt, ist aber mit etwas Übung schneller und präziser als jede grafische Oberfläche.

Das Terminal 02

Das Terminal ist das Fenster, in dem man Befehle eingibt. Ein Befehl besteht meist aus einem Namen (z.B. ls), gefolgt von optionalen Flags (z.B. -l) und Argumenten (z.B. einem Ordnernamen). Nach dem Drücken von Enter führt das System den Befehl aus und zeigt das Ergebnis direkt darunter an.

BEISPIEL
$ ls -l dokumente/
Aufbau des Handbuchs 03

Das Handbuch ist in 13 Kapitel unterteilt, jedes mit mehreren Befehlen. Jeder Befehl wird gleich aufgebaut erklärt: eine kurze Beschreibung, die Syntax, die wichtigsten Flags in einer Tabelle, ein realistisches Beispiel mit erwartetem Output und, wo hilfreich, ein zusätzlicher Hinweis. So findet man sich in jedem Kapitel sofort zurecht.
